## 📦 Local Development

This portfolio is built with **React + Vite**.

### Prerequisites
- Node.js 16+
- npm or yarn

### Setup
```bash
# Clone the repository
git clone https://github.com/okalangkenneth/KensBackendPortfolio.github.io.git
cd KensBackendPortfolio.github.io

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The site will be available at `http://localhost:5173`

## 🚀 Deployment

This site is deployed on **GitHub Pages** with a custom domain.

### Build & Deploy
```bash
# Build the project (outputs to /docs folder)
npm run build

# Commit and push
git add .
git commit -m "Update portfolio"
git push origin master
```

GitHub Actions automatically deploys from the `master` branch, `/docs` folder.
